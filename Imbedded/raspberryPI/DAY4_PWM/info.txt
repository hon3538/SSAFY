기존 GPIO는 3.3V와 0V 만 출력할 수 있었다.

만약 3.3V의 50% 전압을 출력하고 싶다면?
-> 임베디드 보드(디지털 장치)가 아날로그 형식으로 표현하는 방법
1과 0을 빠르게 반복한다.
1과 0을 각각 50% 비율로 반복하면 50%수준의 전압을 인가할 수 있다

# PWM 펄스폭 조절
빠른속도로 한 주기 안에 1과 0의 비율을 조절하여 마치 아날로그 신호인 것처럼
보이게 한다.

# 코딩 방법
1) 디지털 출력API(함수)를 이용해서 직접 빠르게 스위칭하는 코드 작성
-> sleep() 함수 이용해서 만들 수도 있고, Timer 함수를 이용해서 만들 수도 있다
2) 아날로그 출력API를 이용해서 코드 작성

0) HARDWARE 적으로 디지털 출력핀만 존재하는 경우
1) PWM 기능 자체가 없는 경우

한 주기 안에서 high의 비율을 듀티비라고 한다
-> 전압이 80%(high 80%, low 20%) 인경우 듀티비 80%이라고 한다

# 모터
-> 전기를 이용하여 회전력을 발생시키는 장치
1. DC 모터
:가장 기본, 간단, pin 2개로 제어(vcc/gnd), 시계/반시계
2. 서보 모터(DC+센서)
:정밀한 각도제어 서보모터(SG90)-> 0~180도 각도제어
3. 스태핑 모터
:

정확한 각도를 위한 "엔코더 모터"
느리지만, 파워를 위한 "기어드 DC모터"
높은 수명을 위해 브러쉬를 제거한 "브러쉬리스 모터"

# 모터 제어시 주의사항
-> 12v or 9v 높은 전압 요구
-> 임베디드 보드는 default 3v
-> 외부전원 공급
-> 또한 바로 최대속력으로 돌리면 무리가기때문에 천천히 속도변화줘야함 
-> gpio 전류는 20mA이므로 직접 모터에 연결하기는 힘이 부족하다
-> gpio는 base 신호용으로 사용하고 외부전원을 따로 연결한다
