
priority queue 를 사용해서 중간에 있는 사람을 업데이트하고 싶다면?
그냥 해당 사람의 값을 업데이트 해주고 pq에 넣어준다.
또한, 업데이트 한 값을 해당 사람의 id에 저장해준다.

그럼 pq에 같은 사람이 2명이 되겠지만, 어차피 해당 사람이 
이 pq에서 나온적이 있는지 (exist 변수)와 해당 사람의 데이터가 최근
업데이트된 데이터가 같은지(recent 변수)를 비교하여

일치한다면 꺼내고 exist=0으로 해준다.

그럼 다음번에 같은 사람이 나올때 exist=0 즉, 해당 pq에 존재하지 않는 사람이므로
무시하고 continue 한다.
